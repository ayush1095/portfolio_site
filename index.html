Ant Colony Optimisation

In computer science and operations research, the ant colony optimization algorithm (ACO) is a probabilistic technique for solving computational problems which can be reduced to finding good paths through graphs. This project aims to reach a optimal solution to the well known Travelling Salesman(TSP) problem using ACO.
